# ⏰ Relógio Digital

Aplicação web desenvolvida com HTML, CSS e JavaScript para exibição dinâmica de horário em tempo real.

## 🚀 Deploy
🔗 https://minimal-clock-pi.vercel.app/

---

## 📌 Funcionalidades

- Exibição da hora em tempo real
- Atualização automática dos segundos
- Interface moderna e responsiva
- Manipulação dinâmica de data e hora com JavaScript

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
